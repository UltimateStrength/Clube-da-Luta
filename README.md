# 🥊 Clube da Luta

Site temático baseado no filme *Clube da Luta*, iniciado como projeto prático durante um curso da **Alura**. O projeto ficou incompleto — o foco principal foi na implementação da lógica de alternância entre dark mode e light mode.

## ✨ O que está implementado

- **Toggle dark/light mode** — alternância de tema via checkbox customizado, com lógica em JavaScript e estilos separados por modo
- Estrutura base de layout: header com logo, área de conteúdo e footer
- CSS organizado com reset (normalize.css) e folhas de estilo separadas por responsabilidade

## 🚧 Status

Projeto **inacabado**. A maior parte do conteúdo ainda está marcada como `To Be Defined` — o objetivo era treinar a estrutura de um site temático durante o curso, não necessariamente finalizá-lo.

## 📁 Estrutura

```
Clube-da-Luta/
├── index.html      # Estrutura da página
├── style.css       # Estilos principais + dark/light mode
├── main.js         # Lógica do toggle de tema
└── assets/         # Logo e imagens
```

## 🛠️ Tecnologias

- HTML5 / CSS3
- JavaScript (vanilla)
- [normalize.css](https://cdnjs.cloudflare.com/ajax/libs/normalize/8.0.1/normalize.min.css)

## 📌 Contexto

Projeto de curso (Alura) — o objetivo era praticar estruturação de layout e manipulação de DOM. A parte mais relevante que ficou foi a implementação do sistema de dark/light mode com toggle customizado.
